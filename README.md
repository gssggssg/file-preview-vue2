# file-preview-vue2

适用于Vue2的文件预览组件

## 安装

```bash
npm i file-preview-vue2
```

## 使用方法

- 引入

```javascript
import FilePreviewVue2 from "file-preview-vue2";

Vue.use(FilePreviewVue2);
```

- 使用

```vue
<!-- 无需import组件，直接全局使用即可。-->
<file-preview-vue2 />
```
